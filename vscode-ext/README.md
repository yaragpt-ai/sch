# TÜRK Dili VS Code Extension

## Kurulum (30 saniye)

1. VS Code'u aç
2. `Ctrl+Shift+P` tuşlarına bas
3. **"Developer: Install Extension From Location..."** yaz
4. Bu klasörü seç: `vscode-ext`
5. VS Code **"Reload"** desin, tıkla

Bu kadar! Artık `.turk` dosyaları renkli görünecek.

## Kaldırma

1. `Ctrl+Shift+P` → **"Extensions: Show Installed Extensions"**
2. **"TÜRK Language Support"** bul
3. Sağ tık → **Uninstall**
